# Security Policy

Hewn is a local CLI wrapper — no hosted service, no network endpoints of
its own. It invokes the `claude` binary with extra flags.

## Reporting a vulnerability

Please do not open public issues for security-sensitive problems.

- Use GitHub's private vulnerability reporting flow if enabled.
- Or contact the maintainer privately through the repository owner's
  public contact channel.

When reporting, include:

- affected file or component
- impact
- reproduction steps
- suggested mitigation if you have one
