# Code of Conduct

## Our standard

Hewn is a technical project. Productive discussion requires directness,
evidence, and respect.

Participants are expected to:

- critique ideas, benchmarks, and implementations without personal attacks
- make claims that can be traced to code, runs, docs, or sources
- assume good faith while still being technically rigorous
- avoid harassment, discrimination, dogpiling, and deliberately hostile behavior

## Unacceptable behavior

This includes:

- personal insults or intimidation
- sustained antagonism after a boundary has been set
- publishing private information without permission
- discriminatory language or targeted harassment
- intentionally derailing technical threads

## Enforcement

Project maintainers may remove comments, lock discussions, or ban contributors
who repeatedly violate this code of conduct.

For conduct issues, contact the maintainer through GitHub or the email address
associated with the repository owner profile.
